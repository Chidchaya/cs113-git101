เวลาประกาศตัวแปร ให้ใส่ค่าเข้าไปเลยดีกว่าประกาษเป็นตัวแปรเปล่าๆ มันจะเปลืองพื้นที่ใน memory 
เวลา เขียนพารามิเตอร์ของฟังก์ชัน เราจะเขียนเป็น int x หรือ type ตามด้วยตัวแปร เป็นการประกาศตัวแปรใหม่เสมอ
ในฟังก์ชัน ถ้ามีตัวแปร ที่ประกาศชื่อซ้ำกัน มันจะ copy ตัวแปรด้านนอก เข้ามาคิดในฟังก์ชัน แต่ตัวแปรภายนอกที่copyมา จะไม่เปลี่ยนค่าใดๆทั้งสิ้น 
ถึงแม้ในฟังก์ชัน จะเลี่ยนค่าตัวแปรยังไง แต่ตัวแปรนอกที่ copy เข้ามา ก็จะยังมีค่าเท่าเดิม เรียกว่า pass by value
ถ้ามีฟังก์ชันต่อไป เวลาจะหาตัวแปรที่เราเคยประกาศมันจะไม่ใช่ตัวแปรเดิมที่ที่ทำกับฟังก์ชันก่อนหน้านี้ แต่มันจะไปหาตัวแปรน้้น จาก grobal และเปลี่ยนค่า
ตัวแปร grobal ที่ประกาศไว้แล้ว ให้เปลี่ยนค่าทันที มันจะไม่ได้ copy ค่าไว้ เหมือนกับฟังก์ชันแรก ที่ประกาศใน main หลังฟังก์ชัน
