วิธีการเริ่มต้นเขียนโปรแกรมในภาษาซี เริ่มบรรทัดแรกด้วยการใช้ #include <stdio.h> ย่อมากจาก standrad input output 
การประกาศ mainของโปรแกรมว่า จะให้รีเทิร์นค่าอะไรออกมา เพราะภาษาซีจะมีการรเทิร์นค่าเสมอ 
ถ้าประกาศว่า int main() นี่คือการประกาศว่า ให้ main เป็น int คือจะต้องรีเทิร์นออกมาเป็นค่า int ซึ่งก็คือการ ใส่ return 0; 
จริงๆแล้วถ้าไม่ใส่ก็ไม่เกิดปัญหา แต่ก็สมควรที่จะใส่ก็จะใส่ เพราะโปรแกรมจะได้สมบูรณ์และรู้ว่าจบโปรแกรมแล้ว 
และการประกาศตัวแปรที่ถูกต้องและเหมาะสม มีอยู่หลายชนิด ได้แก่
char a[] = character  => 'A' คือพวกตัวอักขระต่างๆ **ใน c จะอ่านตัวอักขระเป็น ascii code เวลาปริ้น ใช้ %c (size of char is 1 byte)
int a = integer  => 10 คือ  เก็บค่าจำนวนเต็ม ก็คือพวกตัวเลขที่ไม่มีทศนิยม  เวลาปริ้น ใช้ %d (size of int is 4 byte)
float a = floatNumber => 3.14 คือ  เก็บค่าจำนวนจริง หรือเรียกง่ายๆว่า ทศนิยม  เวลาปริ้น ใช้ %f (size of float is 4 byte)
double a = doubleNumber => 3.14114 คือ  เก็บค่าจำนวนจริง เหมือน float แต่ว่า จะเก็บทศนิมได้ยาวและเยอะกว่า  เวลาปริ้น ใช้  %lf (size of double is 8 byte)
long = longNumber => 100000000000 คือ เก็บค่าจำนวนเต็ม เหมือน int แต่ว่าจะเก็บค่าได้เยอะกว่า  เวลาปริ้น ใช้ %ld (size of long is 4 byte)
long long = longlongNumber => 100000000000000000 คือ เก็บค่าจำนวนเต็ม เหมือน int แต่ว่าจะเก็บค่าได้เยอะกว่า int และ long เวลาปริ้น ใช้ %lld
(size of longlong is 8 byte)
unsign integer คือ เก็บได้ไม่สิ้นสุด เก็บได้เยอะกว่า long และ longlong
int/int ได้ int 
float/flat ได้ float 
float/int ได้ float เพราะ float ใหญ่กว่า int 
ex. float a = 20/5 ถ้าเรามองจากสมการด้านขวา เราจะตอบเป็นจำนวนเต็ม นั่นคือ 4 แต่พอได้คำตอบแล้ว เราต้องกลับไปมองสมการด้านซ้ายด้วย ว่าประกาศตัวแปรเป็น type อะไร
